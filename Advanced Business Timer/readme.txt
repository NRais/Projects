Installation:

- Download the Java .jar application AdvancedBusinessTimer-Jar-3.0.jar.

(If you do not have a JRE installed)

 - Download the newest Java JRE from the Java website (see http://www.wikihow.com/Install-Java)
 - Install Java
 - Follow the steps below

(If you have a (Java) JRE 7+ installed)

 - Extract the Jar exectuble from the archive
 - Double click the Jar to run


NOTE: Antivirus software may incorrectly block this programs installation. If errors persist temporarily disable any antivirus' that are causing issues.
NOTE: Avast antivirus is known to run "deep screen analysis" on this application. When avast is analyzing please do not do anything and wait until it has finished.
